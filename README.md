## 🧑‍💻 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/techie084/Techie08.git
```

2. Run this command to install dependency:

```npm
npm install
npm run dev
```

3. Open your browser and visit <http://localhost:5173>
